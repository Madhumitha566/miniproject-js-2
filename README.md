#  Personal Budget Tracker

A clean, responsive, and intuitive web application for tracking and managing personal financial transactions (income and expenses). This project is built using **vanilla HTML, CSS, and JavaScript**, ensuring high performance and zero external dependencies. Data persistence is achieved using the browser's **Local Storage** API.

## âœ¨ Features

* **Balance Summary:** Real-time display of **Total Income**, **Total Expenses**, and **Net Balance** at the top of the interface.
* **CRUD Operations:** Full functionality to **Create**, **Read**, **Update** (Edit), and **Delete** transactions.
* **Data Persistence:** Uses **Local Storage** to save all entries, ensuring data remains available across browser sessions.
* **Dynamic Filtering:** Easily switch between displaying **All**, **Income**, or **Expense** entries using radio button filters.
* **Responsive Design:** The layout is fully responsive, providing an excellent user experience on both **desktop** and **mobile devices**.
* **Clean Interface:** A modern, minimal design emphasizing readability and ease of use, with clear visual distinction between income and expense entries.

##  Technology Stack

* **HTML5:** Provides the structure and semantic elements for the application.
* **CSS3:** Handles all styling, including flexbox/grid for layout, and media queries for responsiveness.
* **JavaScript (ES6+):** Manages all application logic, including form handling, DOM manipulation, data filtering, and Local Storage communication.

## Getting Started

To run this project locally, simply follow these steps:

1.  **Download/Clone:** Download the project files (`index.html`, `style.css`, `script.js`).
2.  **Open in Browser:** Open the `index.html` file directly in any modern web browser (e.g., Chrome, Edge).
